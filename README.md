# 14-fullstackjs-idea-KaChr

## Quickstart

1. Clone this repo
2. Install dependencies: `cd client/ && npm i && cd ../server && npm i`
3. In the `server` folder, do `cp .env.example .env`
4. Now start `server` and the client `client`:

```bash
  $ cd client && npm start
  # In a different tab
  $ cd server && npm start
```
